# Responsive Website using CSS Media Queries

## Overview

This project demonstrates how to convert a desktop-only webpage into a mobile-friendly and responsive website using CSS Media Queries. The responsive design automatically adjusts the layout, typography, images, and navigation for different screen sizes, ensuring an optimal viewing experience across desktops, tablets, and smartphones.

## Objective

The objective of this project is to understand and implement responsive web design principles by using CSS Media Queries. The website adapts its layout based on the user's device without modifying the HTML structure.

---

## Features

* Responsive layout using CSS Media Queries
* Mobile-friendly navigation menu
* Flexible layouts using CSS Flexbox
* Responsive images that scale within containers
* Reduced font sizes for smaller screens
* Stacked sections for better mobile readability
* Overflow and horizontal scrolling issues resolved

## Responsive Breakpoints

### Tablet View (≤768px)

* Navigation menu stacks vertically
* Hero section changes from row to column layout
* Font sizes are reduced
* Images resize automatically
* Content is center aligned

### Mobile View (≤480px)

* Buttons become full width
* Images fit within the screen
* Navigation becomes fully stacked
* Overflow is removed
* Footer adapts for smaller screens

---

## How to Run

1. Open the project in Visual Studio Code.
2. Open `index.html` using Live Server.
3. Open Chrome DevTools.
4. Enable the **Device Toolbar** (`Ctrl + Shift + M`).
5. Test the layout on different screen sizes.
